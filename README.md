# Historical Record Indexes

This repository stores systematic indexes of various historical registers (such as parish records, censuses, and notarial acts) to facilitate genealogical and historical research. 

To ensure the data reaches as many researchers as possible, the indexes found in this repository are also published and fully searchable on [Geneanet](https://www.geneanet.org/fonds/?deposant=evangalli).

## Data Structure

Currently, the files are provided "as is" and their structure (columns, formats, etc.) may vary depending on the type of document and when it was transcribed. 

A future effort is planned to standardize the structure and formatting of all files across the repository. This will make the data easier to process, merge, and analyze.
